# LandingEye

## Personal Portfolio Website

A modern, interactive portfolio showcasing my work, skills, and projects with a unique 3D-animated landing experience.

## Features

- **Interactive 3D Landing Animation** - Eye-catching Three.js animation on the main page
- **Multi-Language Support** - Seamlessly switch between English and French
- **Responsive Design** - Beautiful layouts that work on all devices
- **Project Showcase** - Dynamic carousel displaying GitHub projects and 3D models
- **Contact Information** - Easy-to-find contact details and social links
- **Technology Stack Display** - Visual representation of skills and technologies

## Project Structure

```
LandingEye/
├─ index.html          # Main HTML structure
├─ style.css          # Styling with nested CSS
├─ main.js            # Three.js animations and interactions
├─ translator.js      # Language switching functionality
├─ languages.js       # Translation data (EN/FR)
├─ githumrepofetch.js # GitHub API integration
├── img/               # Image assets
└── videos/            # 3D model showcase videos
```

## Getting Started

1. Clone the repository
2. Open `index.html` in a modern web browser
3. Explore the interactive sections using the corner buttons
